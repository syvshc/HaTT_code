# README

This repository contains the code of [arxiv:2410.04385](https://arxiv.org/abs/2410.04385)

The `daas` directory includes the restructured code for algorithms from [Randomized Algorithms for Rounding in the Tensor-Train Format](https://epubs.siam.org/doi/10.1137/21M1451191), utilizing data structures from the [TT-Toolbox](https://github.com/oseledets/TT-Toolbox).

The `hadamard` and `test` directories contain the implementation of algorithms from our paper, along with the numerical experiences' codes in Section 4.
